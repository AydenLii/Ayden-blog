    Home Assistant安装指南

    # Home Assistant安装指南

    # 准备工作

    • 树莓派 4 (opens new window)（树莓派 3 也可以，如果你身边有一个）。
    • Raspberry Pi 4 (opens new window)电源 (opens new window)或Raspberry Pi 3 电源
    • 微型 SD 卡 (opens new window)。建议使用 32 GB 或更大的卡。
    • SD 读卡器。这已经是大多数笔记本电脑的一部分，但如果您没有，您可以购买独立的 USB 适配器。 (opens new window)品牌无所谓，选最便宜的就行。
    • 以太网电缆 (opens new window)。安装时需要。安装后，家庭助理可以使用 Wi-Fi，但以太网连接更可靠，强烈推荐。

    # 将映像写入启动媒体

    1. 将 Home Assistant 启动媒体（SD 卡）连接到您的计算机

    2. 下载并启动Balena Etcher (opens new window)。（您可能需要在 Windows 上以管理员权限运行它）。

    3. 选择写入方式，这里建议从Github (opens new window)（或HAchina (opens new window)）上下载后选择文件烧入。

    4. Balena Etcher 现在将下载图像，完成后单击“选择目标”

    5. 选择要用于 Raspberry Pi 的 SD 卡

    6. 点击“Flash！” 开始写入img。

    7. 当 Balena Etcher 完成图像的写入后，您将收到此确认信息

    # 启动你的树莓派

    1. 插入您刚刚创建的启动媒体（SD 卡）。
    2. 为网络连接以太网电缆。
    3. 连接电源线。
    4. 在您的桌面系统的浏览器中，几分钟内您将能够在homeassistant.local:8123 (opens new window)上访问您的新家庭助理。
    • 如果您运行的是较旧的 Windows 版本或具有更严格的网络配置，您可能需要通过homeassistant:8123 (opens new window)或http://X.X.X.X:8123（将 XXXX 替换为您的 Raspberry Pi 的 IP 地址）访问 Home Assistant。
